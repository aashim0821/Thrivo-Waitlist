# Security Policy

## Supported Versions

We actively support and patch the latest release of Thrivo Waitlist. 

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of our services and users seriously. If you find any vulnerabilities or security issues in this repository, please do not open a public GitHub issue. Instead, report them privately.

To report a vulnerability:
1. Email us at **security@thrivo.com** with details of the vulnerability.
2. Provide a detailed description of the issue, steps to reproduce, and potential impact.
3. We will acknowledge receipt of your report within 48 hours and work with you to patch it quickly.

Thank you for helping keep Thrivo secure!
